Focal Loss는 CrossEntropy의 클래스 불균형을 개선하기 위해 어렵거나 오분류되는 경우에 대하여 더 큰 가중치를 주는 방법입니다.  
위 코드는 Focal Loss에 Calibration의 기법 중 하나인 Label Smoothing 기법을 더 쉽게 적용시키기 위해 재해석한 코드입니다.
